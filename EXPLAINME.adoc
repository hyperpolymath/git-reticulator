= git-reticulator -- EXPLAINME

== Semantic lattice builder for git repositories in AffineScript

[quote, Cargo.toml]
____
Semantic lattice builder for git repositories in AffineScript.
____

git-reticulator builds a lattice (partial order) over commits and refs in a
git repository, using AffineScript's affine-type semantics for safe traversal.
See `src/lattice/` for the lattice construction and `src/api/` for the REST
interface.

**Caveat:** The README is currently empty beyond the title. The claim above
comes from `Cargo.toml` -- the lattice module exists but the project is early.
